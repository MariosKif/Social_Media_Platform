# Excel-like Spreadsheet Website

A website built with Astro that provides an Excel-like spreadsheet interface.

## Features

- Excel-like spreadsheet interface
- Cell editing
- Row and column headers
- Context menu
- Column and row resizing
- Sorting and filtering
- Cell merging
- Formula support

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The site will be available at `http://localhost:4321`

### Build

```bash
npm run build
```

### Preview

```bash
npm run preview
```

## Technologies

- Astro - Web framework
- React - UI library
- Handsontable - Spreadsheet component
